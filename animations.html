PyGameUI Documentation: Animations

←Back: Buttons

Any UI element (text, inputs and general elements) can be moved around using a sweet of diffrent functions that are listet on the bottom of this page.

Function Explanation Arguments
move_to()Moves the element to provided x,y cordinates (instantly, no animation)x, y
move()Moves the element by x, y steps.x, y
flow()Moves the element between two points. The more iterations the slower the animation.position1(x,y), position2(x,y), iterations
jump()Jumpes the element between two points. The more iterations the longer between the jumps.position1(x,y), position2(x,y), iterations
flow_toggle()Toggles flowing.None
jump_toggle()Toggles jumping.None
update()Every time update is called, 1 iteration for jump() and flow() is done.None
